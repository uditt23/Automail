## Run Locally

Clone the project

```bash
  git clone https://github.com/uditt23/Automail
```

Go to the project directory

```bash
  cd gmail
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

* [NodeJS](https://nodejs.org/en/docs)
